## üé® Animation Details

### **Tech Grid Dancing Effect**
```css
@keyframes techFloat {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}
```
- **Duration**: 3 seconds infinite loop
- **Staggered Timing**: Even items delayed by 1.5s
- **Effect**: Creates wave-like floating motion

### **Accordion Animations**
- **Smooth Expand/Collapse**: CSS max-height transitions
- **Icon Rotation**: 180-degree chevron rotation
- **Hover Effects**: Background color transitions

### **Modal Animations**
