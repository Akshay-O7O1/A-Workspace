Ternary Operator in JavaScript

condition ? valueIfTrue : valueIfFalse

Basic Ternary Operator

The ternary operator is a shorthand for if-else statements

let result = age >= 18 ? "Adult" : "Minor";

Nested Ternary Operators

Multiple conditions can be chained together

let grade = score >= 90 ? "A" : score >= 80 ? "B" : score >= 70 ? "C" : "F";

Ternary with Functions

Ternary operators can call different functions

isLoggedIn ? showDashboard() : showLoginForm();

Ternary for Default Values

Using ternary to provide fallback values

let username = user.name ? user.name : "Guest";
