Comparison Operators in JavaScript

Equal (==)

Compares values with type coercion

5 == "5" // true (string converted to number)

Strict Equal (===)

Compares values without type coercion

5 === "5" // false (different types)

Not Equal (!=)

Checks if values are not equal with type coercion

5 != "6" // true

Strict Not Equal (!==)

Checks if values are not equal without type coercion

5 !== "5" // true (different types)

Greater Than (>)

Checks if left value is greater than right value

10 > 5 // true

Less Than (<)

Checks if left value is less than right value

3 < 8 // true

Greater Than or Equal (>=)

Checks if left value is greater than or equal to right value

10 >= 10 // true

Less Than or Equal (<=)

Checks if left value is less than or equal to right value

5 <= 7 // true
